## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/MdSaifulIslamRafsan/Sales-Dashboard.git
   cd Sales-Dashboard
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Run the development server:**
   ```sh
   npm run dev
   ```

> This project does _not_ require any `.env` file setup.
